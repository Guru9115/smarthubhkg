# ✈ SmartHubHKG — GuruSales v2

Flight ticket sales accounting web app with PDF ledger download.

## 🆕 What's New in v2
- **⬇ Ledger PDF Download** — Professional branded PDF with full debit/credit statement
- **⬇ CSV Export** — Export any agent ledger to spreadsheet
- **🦅 NAC Crane** portal added — book.nac.crane.aero
- **12 booking portals** with quick-link buttons
- Improved sidebar with portal section
